# Week 3 – Predictive Modeling & Forecast Insights

## Objective
Forecast monthly application counts using regression-based models.

## Models Used
- Linear Regression
- Random Forest Regressor

## Evaluation Metrics
- RMSE
- MAE
- MAPE
- R²

## Forecast Horizon
Next 3 months

## Key Results
Random Forest outperformed Linear Regression with lower RMSE and MAE.

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run:
   python predictive_model.py
